      This repository is a collection of practice scripts and mini‑projects focused on Python libraries that are widely used in Artificial Intelligence (AI) and Machine Learning (ML).
The goal is to build a strong foundation in programming, numerical computation, data handling, and model development through hands‑on coding exercises.

🎯 Objectives
Practice Python libraries step by step.

Strengthen understanding of AI/ML concepts through coding.

Explore data manipulation, visualization, and modeling workflows.

Prepare for academic assignments, internships, and real‑world projects.
